## Step 0: Install

Install packages, Run the following commands:
1. npm install
2. cd ios
3. pod install
4. cd ..

## Step 1: Start the Metro Server

To start Metro, run the following command from the _root_ of your React Native project:

1. npm start
2. i - for ios
3. a - for android

# OR

## Step 2: Start your Application

### For Android
npm run android

### For iOS
npm run ios
